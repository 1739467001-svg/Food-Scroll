🎯 项目简介

食笺是一个云原生美食菜单管理系统，专为中餐馆设计。

采用中式传统美学设计，将现代科技与中华饮食文化完美融合。

顾客扫码即可浏览精美菜单，餐厅老板实时管理菜品，数据云端同步。

React 18 Node.js Docker 云原生

✨ 核心亮点

  • 3小时完成从0到生产级部署
  • 响应式设计，完美适配手机/平板/桌面
  • JWT认证 + bcrypt加密，安全可靠
  • Docker容器化，一键部署6大云平台
  • 5份技术文档，代码详细注释
  • 完全开源，MIT协议
